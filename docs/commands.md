# Command Reference

## Configuration
- `/adopt name:<name>`: Create your Kiby profile.
- `/revive`: Restore your most recently fallen Kiby.
- `/sleep schedule set timezone:<IANA> start:<HH:mm> duration_hours:<1-9>`
- `/sleep schedule view`
- `/sleep schedule clear`

## Care
- `/feed`
- `/pet`
- `/play`

## Economy
- `/shop list`
- `/shop buy item:<item> quantity:<n>`
- `/inventory`
- `/use item:<item>`

## Progression
- `/daily`
- `/quests view`
- `/quests claim`

## Information
- `/info`
- `/cooldowns`
- `/leaderboard`
- `/help`
- `/ping`

## Developer
- `/system subject:<text> body:<text>` (developer-only)
