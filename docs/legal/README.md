## Version Archive

- Terms of Service versions are stored in [`terms/`](terms/)
- Privacy Policy versions are stored in [`privacy/`](privacy/)

When updating a policy:

1. Add a new versioned file (for example, `v2026-04-10.md`).
2. Update the corresponding `current.md` file to point at the new version.
3. Keep prior version files in place for historical access.

- Publish Terms + Privacy links in app metadata and onboarding UI.
- Verify disclosures align with actual data flows and vendors.
- Confirm payment, cancellation, and refund terms match platform billing rules.
- Confirm GDPR/UK GDPR and CCPA/CPRA rights request flows are operational.
- Confirm COPPA/children handling is accurate for your audience configuration.
